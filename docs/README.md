# 文档索引

这组文档承载仓库协作所需的技术说明。原则是少而清楚，避免把说明拆成很多重复页面。

## 推荐顺序

1. [handbook.md](handbook.md)
2. [quickstart.md](quickstart.md)
3. [architecture.md](architecture.md)
4. [contracts.md](contracts.md)
5. [troubleshooting.md](troubleshooting.md)

## 文档说明

- `handbook.md`：项目总手册，讲清项目目标、业务逻辑、模块和工作流。
- `quickstart.md`：第一次跑项目怎么开始。
- `architecture.md`：项目为什么这样分层。
- `contracts.md`：关键文件、目录和产物之间怎么对接。
- `troubleshooting.md`：常见故障和恢复方法。

如果你想先整体理解项目，先读 `handbook.md`，再看 `quickstart.md` 和 `architecture.md`。
