# 常见问题

这份文档是给技术同学的排障入口，目标是让第一次接触项目的人能自己定位问题，而不是卡在环境和路径上。

## 常见问题

### 1. MinerU 跑不起来

- 先确认 `MINERU_TOKEN` 是否可用。
- 再确认输入文件类型和路径是否正确。
- 如果是批量任务，先用单文件验证。

### 2. ChinaMoney 下载失败

- 先确认是否按 skill 里的方式建立会话。
- 再检查网络、Cookie 和浏览器自动化依赖。
- 如果是 URL 问题，先用官方列表页确认内容是否存在。

### 3. 解析结果不完整

- 先看原始 PDF 是否扫描质量太差。
- 再看附注范围是否定位正确。
- 如果正文和附注混在一起，先做标准化，不要直接进分析。

### 4. 分析结果只是草稿

- 这是预期行为。
- 草稿产物是中间层，不是最终交付。
- 正式报告必须在复核后显式生成。

### 5. 路径和编码出错

- 中文路径要统一用 UTF-8 处理。
- 绝对路径优先。
- 不要把运行态数据写到 skill 目录。

## 排障顺序

1. 看输入是否正确。
2. 看环境是否满足。
3. 看中间产物是否生成。
4. 看正式产物是否缺失。
